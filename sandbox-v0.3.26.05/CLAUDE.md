# Pygmalion / C.R.I.S.T.A.L.L. — MVP v0.3.26 "Иммунитет Импульса"

## О проекте

Минимально жизнеспособное пространство опыта. MVP не является продуктом или системой. Это форма наблюдения нематериального человеческого участия без денег, принуждения и оценки.

## Архитектура

- **Фронтенд**: Vanilla JS + HTML + CSS (Liquid Glass Design System)
- **Хранение**: localStorage (автономный режим)
- **Локализация**: ru/en (i18n)
- **Версия**: v0.3.26 "Иммунитет Импульса" и v0.3.26.05 "золотая версия"

## Основные файлы

- `index.html` — главная страница песочницы
- `logic.js` — вся бизнес-логика (5 протоколов)
- `style.css` — стили Liquid Glass
- `assets/i18n/` — переводы (ru.json, en.json)

## 5 Связанных Протоколов

1. **::про.1.П.Л.А.Н.::** — Эмиссия У.Е. (Recognition Units)
2. **::про.2.Т.О.К.-О.Р.А.К.У.Л.-С.::** — Передача благодарности (ro.DAG)
3. **::про.3.К.О.Л.ЛИЦО-ОБЛИК::** — Отражение (репутационный образ)
4. **::про.4.В.Е.С.::** — Репутационный вес
5. **::про.4,5.Б.Б.О.О.С.С.::** — Саморегулирование (Ордена, Отделы, Союзы, Советы)

## Канон "24+4" (Иммунитет Импульса)

- У.Е. эмитированные 20:00–03:59 получают статус "импульс" (28 часов жизни)
- У.Е. эмитированные 04:00–19:59 получают статус "активна" (24 часа жизни)
- Импульсные У.Е. защищены от сгорания до следующего дня
